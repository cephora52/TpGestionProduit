Description du projet
Cette application est une API REST CRUD permettant de gérer des produits.
Elle est développée avec Spring Boot, documentée avec JavaDoc + Springdoc OpenAPI (Swagger UI), et respecte les bonnes pratiques de nommage, d’annotations et de structuration d’un projet moderne.
Elle expose des opérations pour :Ajouter un produit, Consulter la liste des produits, Rechercher un produit, Modifier un produit, Supprimer un produit

Technologies utilisées

Java 17+
Spring Boot 3.x
Spring Web
Spring Data JPA
H2 Database (ou MySQL, selon configuration)
Lombok
Springdoc OpenAPI 2.6.0
Maven
